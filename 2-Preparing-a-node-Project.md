## Go To Terminal

`npm init`

> This is for initializing a node based project 

then give required name and other information 

* package name :
* version :
* description :
* entry point : (index.js)
* test command :
* git repository :
* keywords :
* author :
* license : (ISC)

is this OK ? (yes)

*This will create a file `package.json`*

**Then create an index.js file**

## To Run the Project 

```
"scripts": {
    "dev" : "node index.js"
}
```
`npm run dev`

## To use `import` Statements 

add `"type" : "module"` in `package.json`
